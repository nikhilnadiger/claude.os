# Script Style Guide

All staffroom scripts follow a confirmed format derived from the existing
IR archive (IR1–IR86). This guide covers the format, the three proven hook
types, tone rules, and Hinglish usage.

---

## Script Format (IR / Reel / Short)

Every script must include all four components in this order:

### 1. Title
Punchy, specific, searchable. Written as a question or a strong statement.
Avoid generic titles. The title must work as a YouTube search query AND as
an Instagram hook.

**Good:** "Is teaching really a woman's job?"
**Good:** "IB teacher salary in India — the real numbers"
**Weak:** "About teacher salaries" / "Teaching challenges"

### 2. Description
Used as the YouTube/Instagram caption. Structure:
- Opening line that mirrors the hook (not just the title)
- 3–5 bullet points with the key data or insights from the video
  (emoji bullets — this is the convention in the archive)
- Source attribution: "Based on [X] experiences shared on staffroom"
  where platform data is used
- CTA: "Write your review at thestaffroom.in"
- Follow prompt: "Follow @thestaffroomdotin on Instagram" /
  "Subscribe to @thestaffroom-in on YouTube" — no cadence implied;
  do not write "for regular updates", "for more content", or similar
  unless a specific content schedule has been approved

### 3. Script Table

The script is structured as a three-column table:

```
| Section | Hear (voice) | See (visual) |
| :--- | :--- | :--- |
| Hook | [voice direction] | [visual direction] |
|  | [continuation] | [continuation] |
| Body | [voice] | [visual] |
| Closure | [CTA voice] | [staffroom logo on screen] |
```

**Sections:**
- **Hook** — first 2–3 seconds. Single most important section. See hook
  types below.
- **Body** — the substance: data, story, examples, argument. Multiple
  rows as needed.
- **Closure** — CTA + follow prompt + staffroom logo on screen. Always
  ends with logo visible.

**Voice direction notes:**
- Write voice direction as instructions to the presenter, not as a
  transcript: "speak directly to camera," "pause after the stat"
- Flag Hinglish lines explicitly: mark them as [Hinglish] so the
  presenter knows to deliver in mixed register
- Emotional beats should be noted: [deliver quietly], [build energy here]

**Visual direction notes:**
- Be specific about what appears on screen: text overlays, B-roll
  suggestions, graphics, data visualisations
- Logo at closure: "staffroom logo bottom-right" or "full-screen staffroom
  card" — always specify
- Do not leave visual column blank for key moments

### 4. Sources
List all data sources used in the script. If platform data: "staffroom
review database (X experiences, accessed [month year])". If external:
full URL. Sources are added to the video description on upload.

## Beat Structure for Short-Form Scripts (≤60s)

Short-form scripts (Reels, Shorts) follow a 5-beat internal structure
within the IR table. This ensures front-loaded value, a curiosity gap,
and an emotionally-anchored CTA — the pattern confirmed by performance data.

> **For Meta Ads scripts only — the cliff is named before any beat is written.**
>
> Meta Ads scripts follow a separate brief and gate process (`skills/meta-ad-creation/references/gate-framework.md`). For those scripts, before writing the hook or any beat below, name the cliff: the specific unresolved question the teacher will be left with after seeing the ad. The cliff must require current employment to feel urgent — a job seeker must not be able to feel the same urgency. If this test is not met, the script is not ready.
>
> The cliff beat appears within the first 7 seconds. Tag it with **[CLIFF]** in the script table row where it lands.
>
> This requirement applies to Meta Ads scripts only. For organic YouTube, Instagram Reels, and WhatsApp content, use the beat structure below without modification.

| Beat | Timing | Function |
|---|---|---|
| Opening | 0–3s | Hook — stops scroll, states value fast. Choose one proven type: Authority, Data, or Validation hook (see hook types section below). Must be the strongest line in the script. |
| Reinforcement | 4–7s | Tease or clarify the benefit. Open a curiosity gap — viewer should feel "I need to know the rest of this." |
| Core Value | 8–30s | Deliver key points with data, examples, named organisations/people. Conversational, fast-paced. Must include at least one pattern interrupt (a shift in tone, a surprising number, a direct question). |
| Escalation | 31–50s | Climax or surprising insight. Can include future pacing: "If you're a teacher in X situation, here's what this means for you." |
| Closing | 51–60s | Single CTA tied directly to the viewer's emotional state at the end of the video. Not a generic "like and subscribe" — it must feel like the natural next step. |

**How this maps to the IR table:**
- Hook row = Opening (0–3s)
- First Body rows = Reinforcement + Core Value
- Last Body row = Escalation
- Closure row = Closing CTA

**Key production rules derived from this structure:**
- The hook must work as a standalone sentence — if someone screenshots
  the first frame, it should communicate the value
- The curiosity gap (Reinforcement) must be resolved in the Core Value —
  never open a gap you don't close
- Pattern interrupt examples: cut to a different visual, drop a surprising
  stat, switch from Hinglish to English or vice versa, lower your voice
- The CTA must be earned — if the content was emotionally resonant, the
  closing line builds on that state, it does not reset to "please share"

## The Three Proven Hook Types

These three hook types are confirmed by performance data across the YouTube
and Instagram archive. Always pick one before writing.

### Hook Type 1: Authority Hook
**What it is:** Open with a credible, named authority — a person, an award,
an institution — whose perspective lends instant credibility.

**When to use:** When you have access to a recognised name, award, or
credential that the audience will recognise and respect. Career & Salary and
Teaching Practice pillars.

**Performance evidence:** NTA "National Teacher Award Winner's Take" —
261,000 Instagram views, 3,397 shares. Authority hook drove reach.

**Formula:**
> "[Authority name / credential] said something about [topic] that
> most teachers don't talk about."

**Example opening line:**
> "India's National Teacher Award winner — here's what she actually thinks
> about [topic]."

**Caution:** Do not manufacture false authority. The named person must be
real and the credential genuine. If you don't have a credible authority,
use a different hook type.

---

### Hook Type 2: Data Hook
**What it is:** Open with a specific, surprising, or counterintuitive number
that makes the viewer stop scrolling.

**When to use:** When you have a specific, credible data point that challenges
a common assumption. Platform Data Reveals, Career & Salary, School Workplace
Reality pillars.

**Performance evidence:** IB salary video (4,134 views, 74.49% completion),
IGCSE video (3,146 views, 73.54% completion). Teachers watch all the way
through when the hook promises specific, rare data.

**Formula:**
> "[Specific number] — [brief context]. Most people don't know this."
> OR
> "We asked [X] teachers. [Specific surprising finding]."

**Example opening lines:**
> "₹22,000 a month. That is what 8 years of experience gets you in most
> private schools in India."
> "Only 34% of teachers in our data say their school's management is good.
> Here's what the rest said."
> "74% completion. The IB salary video people can't stop watching."

**Requirements:** The number must be real, sourced, and specific. Round
numbers feel fabricated. "About 30%" is weaker than "34%." If using platform
data, query Neon first to confirm the exact figure.

---

### Hook Type 3: Validation Hook
**What it is:** Open by naming a specific, unspoken experience the audience
lives but rarely hears articulated. The viewer feels immediately seen.

**When to use:** Teacher Wellbeing and School Workplace Reality pillars.
When the goal is sharing over reach — validation hooks drive shares because
teachers forward content that names their reality to people who would
recognise it.

**Performance evidence:** "Teachers Are TIRED" — 35K views, 1,217 shares
(3.5% share rate — highest in the channel). Share rate is the signal for
validation hooks, not view count. "Why Do Teachers Change Schools Frequently"
— mission-aligned, meaningful share engagement.

**Formula:**
> "If you've ever [specific, named experience], this is for you."
> OR
> "[Name the thing no one says out loud]."

**Example opening lines:**
> "18% ne toh ye bhi kaha ki..." [Hinglish — name a specific finding from
> the data that teachers will recognise as true]
> "You stayed back 2 hours. Your family asked why you couldn't just
> switch off."
> "Teachers are tired. Not the kind of tired that sleep fixes."

**The validation hook is not victimhood.** The hook names the experience;
the body must move toward dignity, data, or action — not amplify grievance.
staffroom's brand is Tripadvisor, not a complaints portal.

## Tone Rules

### Voice register
- Warm and direct. Not corporate. Not NGO-earnest. Not clickbait.
- Teacher-to-teacher, not expert-to-student. staffroom speaks alongside
  teachers, not down to them.
- Confident about data, appropriately uncertain about interpretation.
  "Our data shows X" — not "Teachers believe X" unless it's a quote.

### Specificity requirement
Every script must contain at least one concrete, specific detail:
a salary number, a named school type, a real scenario, a direct quote,
or a platform data point. Generic claims ("teachers struggle financially")
without a specific anchor are not acceptable.

### Length norms by format
- Instagram Reel / YouTube Short: 45–75 seconds spoken content
- YouTube long-form: 8–14 minutes for Career & Salary; 5–10 minutes for
  Practice / Wellbeing; no padding to hit a duration target
- WhatsApp broadcast: 3–5 sentences maximum; no script table needed

### What to avoid
- "Leverage," "ecosystem," "at scale," "unlock," "empower" — jargon that
  distances from the teacher audience
- Rhetorical questions used as filler ("Have you ever wondered...?") —
  every question must land on a real answer
- Calls to action that feel guilt-driven ("Your school depends on you")
- Overpromising ("staffroom will fix education") — brand rule

## Hinglish Usage

Hinglish is a deliberate creative tool for staffroom content, not an
afterthought. It signals authenticity and lowers the social distance
between the content and the viewer.

**When to use Hinglish:**
- In the hook, when naming an emotional experience: "18% ne toh ye bhi
  kaha ki..." (from archive) — Hinglish lands harder here than English alone
- In quotes attributed to teachers — deliver in the register the teacher
  would have used
- In the body when making a direct appeal to the viewer
- In closures: "apna experience share karo" alongside the English CTA

**When to stay in English:**
- Data points and statistics — numbers in English are cleaner to read and share
- Titles — for searchability, English titles index better
- School type names: CBSE, IGCSE, IB, government school — keep these in
  standard English

**Grammar note:** Hinglish in this context is conversational, not
written-standard Hindi. It follows spoken register: verb-last structures
are fine, transliterated words are fine. Do not attempt grammatically
"correct" Hindi — it sounds unnatural. Write it as it would be spoken.

**Performance evidence for Hinglish/bilingual:** Meta Ads data directly
confirms this. Kannada-English (KaEn) ad creatives targeting Bengaluru
converted at CPR ₹144 vs ₹279 for English-only — nearly 2x better. Code-
switching content is not just a cultural preference; it is a measurable
performance signal.

## CTA Standards

**Primary CTA (for content driving reviews):**
> "Apna school ka review likho — thestaffroom.in"
> OR
> "Write your review at thestaffroom.in"

**Secondary CTA (for follow/community growth):**
> "Follow @thestaffroomdotin on Instagram" / "Subscribe to @thestaffroom-in on YouTube"

**Never use:**
- "Share an experience" / "Share your experience" — retired; now the wrong form
- "Rate your school" — reduces the act to a rating, misses the brand positioning
- "Join our community" — vague, not action-specific

## Long-Form Script Format (YouTube Episodes)

Long-form scripts expand the IR table into a full outline with a Duration column.

```
**Title:** [title — explicit, includes year for salary content]
**Description:** [full YouTube description — see template below]

**Outline:**
| Section | Hear (voice) | See (visual) | Duration |
| :--- | :--- | :--- | :--- |
| Hook | | | 0:00–0:30 |
| Setup | | | 0:30–2:00 |
| [Section 1] | | | 2:00–5:00 |
| [Section 2] | | | 5:00–9:00 |
| Closure + CTA | | | 9:00–9:30 |
```

**Description template:**
```
[2–3 sentence overview]

What we cover:
0:00 — [Chapter name]
2:00 — [Chapter name]
5:00 — [Chapter name]
9:00 — [CTA]

Key data points:
• [Specific finding with number]
• [Specific finding with number]

Based on [X] teacher experiences shared on staffroom.
Write your review: thestaffroom.in

Sources:
[List all data sources]
```

**Length targets:** 8–14 min for Career & Salary. 5–10 min for Practice / Wellbeing.
No padding to hit a duration — end when the content ends.

**Title rules:** Write as a specific, searchable statement. Include year for salary
content ("IB Teacher Salary in India — 2025"). No "SHOCKING" or "You won't believe"
— the data is strong enough without inflation.

**Thumbnail:** Every long-form video needs a thumbnail — brief it using visual-asset-creation.
